## 📝 Convenciones

### Nombres de Archivos
- Usar MAYÚSCULAS para archivos principales
- Usar snake_case o kebab-case para archivos secundarios
- Extensión `.md` para todos los documentos

### Estructura de Documentos
Todos los documentos deben incluir:
1. **Título principal** (# H1)
2. **Resumen** o introducción
3. **Secciones organizadas** (## H2, ### H3)
4. **Ejemplos de código** cuando aplique
5. **Fecha de última actualización**

### Emojis Estándar
- 📚 Documentación
- 🚀 Despliegue
- 🔧 Configuración
- 📊 Datos/Reportes
- ✅ Completado
- ⏳ En progreso
- ❌ Error/Problema
- 💡 Tip/Nota importante
- 🎯 Objetivo
- 📋 Lista/Checklist
